SW360

мҳӨн”ҲмҶҢмҠӨлҘј нҸ¬н•Ён•ҳлҠ” м ңн’Ҳмқ„ к°ңл°ңн•ҳкі  л°°нҸ¬н•ҳлҠ” кё°м—…мқҙлқјл©ҙ к°Ғ м ңн’Ҳкіј лҰҙлҰ¬мҠӨ лІ„м „л§ҲлӢӨ мӮ¬мҡ©н•ң мҳӨн”ҲмҶҢмҠӨмқҳ лІ„м „, лқјмқҙм„ мҠӨ л“ұмқҳ м •ліҙлҘј мҲҳм§‘н•ҳкі  м¶”м Ғн•ҙм•ј н•ңлӢӨ. мқҙлҘј нҶөн•ҙ кё°м—…мқҖ мҳ¬л°”лҘё мҳӨн”ҲмҶҢмҠӨ м»ҙн”Ңлқјмқҙм–ёмҠӨ нҷңлҸҷмқ„ мҲҳн–үн•  мҲҳ мһҲлӢӨ.

нҠ№нһҲ, NVD (https://nvd.nist.gov/vuln)м—җм„ң нҠ№м • мҳӨн”ҲмҶҢмҠӨ лІ„м „м—җ ліҙм•Ҳ м·Ём•Ҫм җмқҙ ліҙкі  лҗҳм—Ҳмқ„л•Ң, н•ҙлӢ№ лІ„м „мқ„ мӮ¬мҡ©н•ҳкі  мһҲлҠ” м ңн’Ҳмқҙ л¬ҙм—Үмқём§Җ м¶”м Ғмқ„ н•  мҲҳ м—ҶлӢӨл©ҙ, к·ё кё°м—…мқҖ м–ҙлҠҗ м ңн’Ҳм—җ ліҙм•Ҳ нҢЁм№ҳлҘј м Ғмҡ©н•ҙм•ј н•  м§Җ м•Ң мҲҳ м—ҶлҠ” мғҒнҷ©м—җ мІҳн•ҳкІҢ лҗҳкі , к·ё кё°м—…мқҳ м ңн’Ҳл“ӨмқҖ ліҙм•Ҳм·Ём•Ҫм җм—җ к·ёлҢҖлЎң л…ём¶ңмқҙ лҗ  мҲҳ л°–м—җ м—ҶлӢӨ.

мқҙл Үл“Ҝ, мҳӨн”ҲмҶҢмҠӨ м •ліҙлҘј м¶”м Ғн•ҳлҠ” нҷңлҸҷмқҖ кјӯ н•„мҡ”н•ҳлӢӨ. кё°м—…л“ӨмқҖ мқҙлҘј мң„н•ҙ мһҗмІҙ мӢңмҠӨн…ңмқ„ кө¬м¶•н•ҳкұ°лӮҳ, мғҒмҡ© м„ңл№„мҠӨлҘј кө¬л§Өн•ҳм—¬ мӮ¬мҡ©н•ҳкё°лҸ„ н•ңлӢӨ. SW360мқҖ Eclipse мһ¬лӢЁм—җм„ң нӣ„мӣҗн•ҳлҠ” мҳӨн”ҲмҶҢмҠӨлЎңм„ң мҶҢн”„нҠёмӣЁм–ҙ BOMм—җ лҢҖн•ң м •ліҙлҘј мҲҳм§‘ л°Ҹ м¶”м Ғн•ҳкё° мң„н•ң мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳ л°Ҹ м ҖмһҘмҶҢлҘј м ңкіөн•ңлӢӨ.

< https://www.eclipse.org/sw360/ >

мЈјмҡ” нҠ№м§•

SW360мқҖ мӣ№ кё°л°ҳмқҳ UIлҘј м ңкіөн•ҳл©° мЈјмҡ” кё°лҠҘмқҖ лӢӨмқҢкіј к°ҷлӢӨ.

  • м ңн’Ҳм—җ мӮ¬мҡ©лҗң м»ҙнҸ¬л„ҢнҠё м¶”м Ғ
  • ліҙм•Ҳ м·Ём•Ҫм җ нҸүк°Җ
  • лқјмқҙм„ мҠӨ мқҳл¬ҙ кҙҖлҰ¬
  • кі м§Җл¬ё л“ұ лІ•м Ғ л¬ём„ң мғқм„ұ

м„Өм№ҳ

SW360мқҖ лӢӨмқҢкіј к°ҷмқҙ кө¬м„ұлҗңлӢӨ.

  • Frontend : Liferay-(Tomcat-)based portal application
  • Backend : Tomcat-based thrift service
  • Database : CouchDB

Project кө¬мЎ°мҷҖ м„Өм№ҳлҘј мң„н•ҙ мҡ”кө¬лҗҳлҠ” мҶҢн”„нҠёмӣЁм–ҙ л“ұ мһҗм„ён•ң лӮҙмҡ©мқҖ READMEмқҳ Required software л¶Җл¶„м—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ. : https://github.com/eclipse/sw360/blob/master/README.md

SW360мқҖ лӢӨмқҢ м„ё к°Җм§Җмқҳ м„Өм№ҳ л°©лІ•мқ„ м ңкіөн•ңлӢӨ. мӮ¬мҡ©мһҗлҠ” мқҙ мӨ‘ н•ҳлӮҳлҘј м„ нғқн•ҳм—¬ м„Өм№ҳн•  мҲҳ мһҲлӢӨ.

  1. Vagrant (https://www.vagrantup.com/) кё°л°ҳ м„Өм№ҳ : VagrantлҠ” к°ҖмғҒнҷ” мқёмҠӨн„ҙмҠӨлҘј кҙҖлҰ¬н•ҳлҠ” лҸ„кө¬лЎңм„ң sw360vagrantм—җм„ңлҠ” SW360мқ„ н•ң лІҲм—җ Deploy н•ҳкё° мң„н•ң нҷҳкІҪмқ„ м ңкіөн•ңлӢӨ. : https://github.com/sw360/sw360vagrant
  2. SW360мқҳ кө¬м„ұмҡ”мҶҢлҘј к°ңлі„м ҒмңјлЎң м„Өм№ҳн•  мҲҳ мһҲлӢӨ. : https://github.com/eclipse/sw360
  3. DockerлҘј нҶөн•ҙ Deploy н•  мҲҳ мһҲлӢӨ. : https://github.com/sw360/sw360chores

м—¬кё°м„ңлҠ” CentOS 7.6 мӢңмҠӨн…ңм—җ Vagrant кё°л°ҳмңјлЎң м„Өм№ҳн•ҳм—¬ Deployн•ҳлҠ” л°©лІ•мқ„ мҶҢк°ңн•ңлӢӨ. мһҗм„ён•ң мӮ¬н•ӯмқҖ READMEлҘј м°ёкі н•ңлӢӨ. : https://github.com/sw360/sw360vagrant/blob/master/README.md

1) мӮ¬м „ м„Өм№ҳ

vagrant boxм—җ SW360мқ„ м„Өм№ҳн•ҳкё° мң„н•ҙм„ңлҠ” openjdk, VirtualBox л°Ҹ VagrantлҘј м„Өм№ҳн•ҙм•ј лӢӨ. лЁјм Җ openjdk 1.8.0мқ„ м„Өм№ҳн•ңлӢӨ.

$ yum install java-1.8.0-openjdk
$ java -version
openjdk version "1.8.0_191"
OpenJDK Runtime Environment (build 1.8.0_191-b12)вҖқ
OpenJDK 64-Bit Server VM (build 25.191-b12, mixed mode)

VirtualBoxлҘј м„Өм№ҳн•ңлӢӨ.

$ sudo wget https://download.virtualbox.org/virtualbox/rpm/el/virtualbox.repo -P /etc/yum.repos.d
$ sudo yum install VirtualBox-5.2

CentOS 7м—җм„ң VirtualBox м„Өм№ҳ мӢң, вҖңkernel module is not loadedвҖҷ м—җлҹ¬к°Җ л°ңмғқн•  кІҪмҡ°, kernel-develмқ„ м„Өм№ҳн•ҳм—¬ н•ҙкІ°н•ң нӣ„ VirtualBoxлҘј мһ¬м„Өм№ҳн•ңлӢӨ.

$ sudo yum install https://centos7.iuscommunity.org/ius-release.rpm
$ sudo yum install dkms
$ sudo yum install kernel-devel
# reboot
$ sudo /sbin/vboxconfig
$ systemctl status vboxdrv
в—Ҹ vboxdrv.service - VirtualBox Linux kernel module
   Loaded: loaded (/usr/lib/virtualbox/vboxdrv.sh; enabled; vendor preset: disabled)
   Active: active (exited) since Wed 2020-02-19 09:06:02 KST; 20min ago

VagrantмҷҖ vagrant-aws pluginмқ„ м„Өм№ҳн•ңлӢӨ.

$ sudo yum install https://releases.hashicorp.com/vagrant/2.2.6/vagrant_2.2.6_x86_64.rpm 
# vagrant-aws plugin м„Өм№ҳ
$ vagrant plugin install vagrant-aws

к·ёлҰ¬кі , sw360vagrant мҪ”л“ңлҘј Clone н•ңлӢӨ.

$ git clone https://github.com/sw360/sw360vagrant.git

2) Dependency лӢӨмҡҙлЎңл“ң

Vagrant boxлҘј л№Ңл“ңн•ҳлҠ” мӢңк°„мқ„ мӨ„мқҙкё° мң„н•ҙ Dependency Packageл“Өмқ„ лҜёлҰ¬ лӢӨмҡҙлЎңл“ң л°ӣлҠ”лӢӨ.

$ cd sw360vagrant
$ ./download-packages.sh

к·ёлҹ¬л©ҙ лӢӨмқҢмқҳ packageл“Өмқҙ ./shared/package нҸҙлҚ” м•Ҳм—җ лӢӨмҡҙлЎңл“ң лҗңлӢӨ.

  • Liferay 7.2.1 CE GA2 with Tomcat (9.0.17)
  • Postgresql-42.2.9 ODBC client for Java as *.jar file
  • SW 360м—җм„ң н•„мҡ”н•ң 11к°ңмқҳ *.jar нҢҢмқј
  • Thrift 0.11
  • A box images from the Ubuntu 16.04 LTS (xenial-server-cloudimg-amd64-vagrant.box)

3) Base box мғқм„ұ

мқҙм ң лӢӨмқҢ лӘ…л №м–ҙлЎң Base boxлҘј мғқм„ұн•ңлӢӨ.

$ cd generate-box
$ ./generate_box.sh

мқҙ мһ‘м—…мқҖ мҲҳмӢӯ л¶„ мҶҢмҡ”лҗ  мҲҳ мһҲлӢӨ.

4) Box мӢӨн–ү

лӢӨмқҢ лӘ…л №м–ҙлЎң BoxлҘј мӢӨн–үн•ңлӢӨ.

# If you have built a vagrant box from this directory earlier, you will have to destroy it first via
$ vagrant destroy
$ cd ../sw360-single
$ vagrant up

BoxлҘј мӢӨн–үн•ҳл©ҙ liferay, postgresql л°Ҹ couchdbк°Җ кө¬м„ұлҗңлӢӨ. мқҙмғҒм—Ҷмқҙ мӢӨн–үмқҙ лҗ  кІҪмҡ°, https://localhost:8443/ лЎң Liferay нҷ”л©ҙм—җ м ‘к·јн•  мҲҳ мһҲлӢӨ.

5) SW360 Layout Deploy

л§Ҳм§Җл§ү лӢЁкі„лҠ” Liferayм—җм„ң SW360мқҳ Layoutмқ„ Deployн•ҳлҠ” кІғмқҙлӢӨ. мқҙ мһ‘м—…мқҖ м•„м§Ғ мһҗлҸҷнҷ”к°Җ лҗҳм§Җ м•Ҡм•„ кҙҖлҰ¬мһҗк°Җ мҲҳлҸҷмңјлЎң мҲҳн–үн•ҙм•ј лӢӨ. https://localhost:8443/м—җ м ‘к·јн•ҳм—¬ лӢӨмқҢ кі„м •мңјлЎң лЎңк·ёмқён•ңлӢӨ.

  • id : setup@sw360.org
  • pw : sw360fossy

мқҙнӣ„м—җлҠ” лӢӨмқҢ мӮ¬мқҙнҠёмқҳ м•ҲлӮҙм—җ л”°лқј Layout deployлҘј мҲҳн–үн•ңлӢӨ. https://github.com/eclipse/sw360/wiki/Deploy-Liferay7

Deployк°Җ мҷ„лЈҢлҗҳл©ҙ лӢӨмқҢкіј к°ҷмқҖ нҷ”л©ҙмқ„ ліј мҲҳ мһҲлӢӨ.

Basic Workflow

1) License л“ұлЎқ

SW360мқ„ мІҳмқҢ м„Өм№ҳн•ҳл©ҙ лЁјм Җ мһҗмЈј мӮ¬мҡ©н•ҳлҠ” мҳӨн”ҲмҶҢмҠӨ лқјмқҙм„ мҠӨ л“Өмқ„ л“ұлЎқн•ҙм•ј н•ңлӢӨ. лқјмқҙм„ мҠӨ лӢӨмқҢкіј к°ҷмқҖ м •ліҙлҘј нҸ¬н•Ён•ңлӢӨ.

  • Full Name
  • Short Name
  • License Type
  • GPL-2.0 Compatibility (мҳҲ: yes, no)
  • License Text

л©”лүҙ > Licenses > Add LicenseлҘј м„ нғқн•ҳл©ҙ лӢӨмқҢкіј к°ҷмқҙ Create License нҷ”л©ҙмңјлЎң м§„мһ…н•ңлӢӨ.

мқҙмҷҖ к°ҷмқҙ лқјмқҙм„ мҠӨлҘј н•ҳлӮҳм”© мҲҳлҸҷмңјлЎң л“ұлЎқн•ҳлҠ” мқјмқҖ мғҒлӢ№нһҲ мҲҳкі мҠӨлҹ¬мҡё мҲҳ мһҲлҠ”лҚ°, лӢӨн–үнһҲ SW360мқҖ SPDX License ListлҘј н•ң лІҲм—җ Import н•ҳлҠ” кё°лҠҘмқ„ м ңкіөн•ңлӢӨ. л©”лүҙ > Admin < Import SPDX Informationмқ„ нҒҙлҰӯн•ңлӢӨ.

к·ёлҹ¬л©ҙ, кі§ SPDX License Listк°Җ мһҗлҸҷмңјлЎң л“ұлЎқлҗ©лӢҲлӢӨ. л©”лүҙ > Licensesм—җм„ң 338к°ңмқҳ Licenseк°Җ л“ұлЎқлҗң кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

2) Component л°Ҹ Release л“ұлЎқ

SW360м—җм„ң ComponentлҠ” н•ҳлӮҳмқҳ мҶҢн”„нҠёмӣЁм–ҙ лӢЁмң„мқҙлӢӨ. м—¬кё°м—җлҠ” лӢӨм–‘н•ң нҳ•нғңмқҳ мҶҢн”„нҠёмӣЁм–ҙк°Җ н•ҙлӢ№н•  мҲҳ мһҲмңјл©°, к·ё мҳҲлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

  • мҳӨн”ҲмҶҢмҠӨ мҶҢн”„нҠёмӣЁм–ҙ
  • лқјмқҙлёҢлҹ¬лҰ¬
  • 3rd party мҶҢн”„нҠёмӣЁм–ҙ

ComponentлҠ” лӢӨмқҢкіј к°ҷмқҖ м •ліҙлҘј нҸ¬н•Ён•ңлӢӨ.

  • Component Name
  • Main Licenses
  • Categories (мҳҲ: Library, Cloud, Mobile, …)
  • Component Type (мҳҲ: OSS, Internal, InnerSource, Service, Freeware)
  • Default Vendor
  • Homepage URL

ReleaseлҠ” Componentм—җм„ң н•ҳлӮҳмқҳ Versionмқ„ к°ҖлҰ¬нӮӨлҠ” лӢЁмң„мқҙлӢӨ. л”°лқјм„ң н•ҳлӮҳмқҳ ComponentлҠ” м—¬лҹ¬ к°ңмқҳ ReleaseлҘј к°Җм§Ҳ мҲҳ мһҲлӢӨ. ReleaseлҠ” н•ҳлӮҳмқҳ Component н•ҳмң„м—җ мғқм„ұлҗҳм–ҙ кҙҖлҰ¬лҗңлӢӨ.

ReleaseлҠ” лӢӨмқҢкіј к°ҷмқҖ м •ліҙл“Өмқ„ нҸ¬н•Ён•ңлӢӨ.

  • Component Name
  • Version
  • License
  • Download URL
  • CPE ID (мҳҲ: cpe:2.3:a:apache:maven:3.0.4)

мҳҲлҘј л“Өм–ҙ, zlib-1.2.8мқ„ л“ұлЎқн•ҙм•ј н•ңлӢӨл©ҙ, лЁјм Җ Componentм—җ zlibмқ„ лЁјм Җ л“ұлЎқн•ң нӣ„, Releaseм—җ zlib 1.2.8мқ„ л“ұлЎқн•ңлӢӨ. Menu > Components > Add ComponentлҘј м„ нғқн•ҳл©ҙ Create Component нҷ”л©ҙмңјлЎң м§„мһ…н•ҳм—¬ zlibм—җ лҢҖн•ң м •ліҙлҘј л“ұлЎқн•  мҲҳ мһҲлӢӨ.

ComponentлҘј мғқм„ұн•ҳл©ҙ, Components > Releases > Add Releaseм—җм„ң zlib-1.2.8 versionм—җ лҢҖн•ң м •ліҙлҘј л“ұлЎқн•  мҲҳ мһҲлӢӨ.

н•ҳлӮҳмқҳ zlibмқҙлқјлҠ” Componentм—җ 1.2.8кіј 1.2.11 versionмқ„ к°Ғк°Ғмқҳ ReleaseлЎң л“ұлЎқн•ҳмҳҖмқ„ л•Ң, Release Overview нҷ”л©ҙм—җм„ң лӢӨмқҢкіј к°ҷмқҙ 2к°ңмқҳ Releaseк°Җ мЎҙмһ¬н•ҳлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.

SW360мқҖ лӢӨмҲҳмқҳ Component м •ліҙлҘј Import мӢңнӮӨкё° мң„н•ң кё°лҠҘмқ„ м ңкіөн•ңлӢӨ. л©”лүҙ > Admin > Import / Exportм—җ CSV templateм—җ л“ұлЎқмқ„ мӣҗн•ҳлҠ” Component м •ліҙлҘј мһ…л Ҙ нӣ„ Import н•  мҲҳ мһҲлӢӨ.

лӢЁ, мқҙ кё°лҠҘмқҖ 2020л…„ 2мӣ” кё°мӨҖ м•„м§Ғ м•Ҳм •м ҒмңјлЎң лҸҷмһ‘н•ҳм§Җ м•Ҡмқ„ мҲҳ мһҲлӢӨ.

3) Project мғқм„ұ

ProjectлҠ” н•ҳлӮҳмқҳ м ңн’Ҳмқ„ к°ҖлҰ¬нӮЁлӢӨ. мӮ¬м—… мң нҳ•м—җ л”°лқј м ңн’ҲмқјмҲҳлҸ„ мһҲкі , м„ңл№„мҠӨ нҳ№мқҖ мҶҢн”„нҠёмӣЁм–ҙ мқјмҲҳлҸ„ мһҲлӢӨ. Projectм—җлҠ” м ңн’Ҳм—җ мӮ¬мҡ©лҗң Component/ReleaseлҘј л“ұлЎқн•ҳм—¬ кҙҖлҰ¬н•ңлӢӨ.

Project мғқм„ұ мӢңм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ м •ліҙлҘј л“ұлЎқн•ңлӢӨ.

  • Project Name
  • Version
  • Project type (мҳҲ: Product, Customer Project, Service, Internal Project, InnerSource)

л©”лүҙ > Projects > Add ProjectлҘј нҶөн•ҙ ProjectлҘј мғқм„ұн•  мҲҳ мһҲлӢӨ.

ProjectлҘј мғқм„ұн•ҳкі  лӮҳл©ҙ, нҸ¬н•Ён•ҳлҠ” ReleaseлӮҳ н•ҳмң„ ProjectлҘј л“ұлЎқн•ңлӢӨ. л©”лүҙ > Projectsм—җм„ң н•ҙлӢ№ ProjectлҘј м„ нғқн•ҳл©ҙ вҖңLinked Releases and ProjectsвҖқм—җм„ң Linked ProjectsмҷҖ Linked ReleasesлҘј л“ұлЎқн•  мҲҳ мһҲлӢӨ.

лӢӨмқҢмқҖ SuperCalcлқјлҠ” Projectм—җ OpenSSL 1.0.1кіј zlib 1.2.8мқ„ Linked ReleasesлЎң л“ұлЎқн•ң мқҙнӣ„мқҳ нҷ”л©ҙмқҙлӢӨ.

4. ліҙм•Ҳ м·Ём•Ҫм җ кҙҖлҰ¬

SW360мқҖ л“ұлЎқлҗң Releaseм—җ лҢҖн•ҙ ліҙм•Ҳ м·Ём•Ҫм җмқҙ мһҲлҠ”м§Җ мһҗлҸҷмңјлЎң нҷ•мқён•  мҲҳ мһҲлӢӨ. мқҙлҘј мң„н•ҙ SW360мқҖ CVE м •ліҙлҘј мЈјкё°м ҒмңјлЎң мҲҳм§‘н•ҳлҸ„лЎқ мҠӨмјҖмҘҙл§Ғн•ҳлҠ” кё°лҠҘмқ„ м ңкіөн•ңлӢӨ. л©”лүҙ > Admin > Schedule м—җм„ң CVE SEARCH м •ліҙлҘј 24мӢңк°„л§ҲлӢӨ мҲҳм§‘н•ҳлҸ„лЎқ мҠӨмјҖмҘҙл§Ғмқ„ м„Өм •н•  мҲҳ мһҲлӢӨ.

мқҙл ҮкІҢ мҠӨмјҖмҘҙл§Ғмқ„ м„Өм •н•ҳл©ҙ SW360мқҖ м •н•ҙм§„ мӢңк°„м—җ CVE Search мӮ¬мқҙнҠё(https://cve.circl.lu/)м—җм„ң CVE м •ліҙлҘј мҲҳм§‘н•ңлӢӨ. мҲҳм§‘н•ң CVE м •ліҙлҠ” л©”лүҙ > Vulnerabilitiesм—җм„ң нҷ•мқён•  мҲҳ мһҲлӢӨ.

мқҙл ҮкІҢ Vulnerabilities м •ліҙк°Җ мҲҳм§‘лҗң мқҙнӣ„м—җлҠ” мғқм„ұн•ң Projectм—җ ліҙм•Ҳ м·Ём•Ҫм җмқҙ мһҲлҠ”м§Җ мЎ°нҡҢн•  мҲҳ мһҲлӢӨ. мң„м—җм„ң мғқм„ұн•ң SuperCalc Projectм—җм„ңлҠ” 85к°ңмқҳ ліҙм•Ҳ м·Ём•Ҫм җмқҙ ліҙкі лҗң кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

мқҙмҷҖ к°ҷмқҖ л°©лІ•мңјлЎң кё°м—…м—җм„ң к°ңл°ң/л°°нҸ¬н•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙлҘј SW360м—җ л“ұлЎқн•ҳм—¬ кҙҖлҰ¬н•ңлӢӨл©ҙ, мҳӨн”ҲмҶҢмҠӨ м»ҙн”Ңлқјмқҙм–ёмҠӨлҝҗл§Ң м•„лӢҲлқј ліҙм•Ҳ м·Ём•Ҫм җм—җ лҢҖн•ҙм„ңлҸ„ лҰ¬мҠӨнҒ¬лҘј мөңмҶҢнҷ”н•  мҲҳ мһҲлҠ” нҳ•нғңлЎң кҙҖлҰ¬к°Җ к°ҖлҠҘн•ҳлӢӨ.

лҳҗн•ң SW360мқҖ мң„мҷҖ к°ҷмқҖ Web Interface лҝҗл§Ң м•„лӢҲлқј лҢҖл¶Җл¶„мқҳ кё°лҠҘмқ„ REST APIлЎң м ңкіөн•ҳм—¬м„ң FOSSology л“ұмқҳ лӢӨлҘё лҸ„кө¬мҷҖмқҳ м—°лҸҷмқҙ к°ҖлҠҘн•ҳлӢӨ. : https://github.com/eclipse/sw360/wiki/Dev-REST-API
